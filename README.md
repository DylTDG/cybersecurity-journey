# cybersecurity-journey
home-soc-lab

Project Overview

This project showcases a hands-on cybersecurity lab built using virtual machines to simulate real-world attack and defense scenarios. The environment includes a Kali Linux attacker machine and a Windows 10 target system.

Lab Setup

Component	    Description

Attacker	    Kali Linux
Target	      Windows 10
Hypervisor	  VirtualBox
Network	      Internal Network (intnet)

Network Configuration
Kali: 192.168.56.11
Windows: 192.168.56.10

Attack Simulation
Performed network scanning using Nmap
Identified open and filtered ports
Enumerated exposed services

Traffic Analysis
Captured HTTP traffic using Wireshark
Observed request/response behavior
Analyzed TCP communication patterns

Security Configuration
Configured Windows Firewall
Allowed HTTP (Port 80)
Controlled ICMP access for testing

Key Skills Demonstrated
Network configuration
Threat detection basics
Packet analysis
System hardening concepts
